# Takashi-Creater.github.io

GitHub Pages 用の個人サイトです。ホームページを入口にして、プロフィール、制作物、更新ログ、連絡先へ移動できる構成にしています。

## この作品について
シンプルな1カラム+サイドバー構成で、どのページからでも迷わず戻れるナビゲーションを用意しています。各ページは同じデザインを共有し、内容だけを差し替えられるようにしています。今後は Works の実績追加や Blog の更新を中心に拡張する想定です。

## ページ一覧
- home.html
- about.html
- works.html
- blog.html
- weather.html
- travel.html
- contact.html

## ローカルプレビュー
起動:
```
./start-server.sh
```

停止:
```
./stop-server.sh
```

ブラウザで開く:
```
http://localhost:8000/home.html
```
